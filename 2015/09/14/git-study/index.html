Git学习笔记

ubuntu下git的安装

  1. 旧版本的ubuntu linux的安装命令：sudo apt-get install git-core
    （我的linux是ubuntu 10.04，需要此条命令才能成功安装）

  2. 新版本的ubuntu只需要：sudo apt-get install git

  3. 以前老版本的Linux上有一个软件交git unit tools（安装命令是sudo apt-get install git并不是安装git代码仓库）。。所以只能使用git-core。之后才改为git。

  4. 基本命令：
    git init
    git add
    git status
    git commit -m “….”

  1. Git是分布式代码管理
  2. Git多平台
  3. Git是Linus（Linux之父的作品，大牛最开始是用C写的）
  4. Git的命令

    Git最好在Linux上使用

  • 建立Git工作目录:mkdir GitPrj
    目录下有文件_git(非常重要，不能轻易删除)
    • Git初始化： git init
    • 在目录下建立一个readme.txt
    • 添加需要提交的文本文件（Git只能管理文本文件，微软的wrod doc格式是二进制文件，不能管理）
      添加命令：git add reame.txt。。git可以添加多个文件，然后一次提交：比如：git add readme_1.txt readme_2.txt…readme_N.txt
    • 提交：git commit -m “提交说明”
    • git status查看git状态
    • git 查看文件先后的区别：git diff readme.txt
    • git 版本回退，首先查看提交日志：git log（完美显示加 –pretty=oneline）
    • HEAD表示当前版本
    • HEAD^表示前一个版本
    • HEAD^^表示前前一个版本
    • 当然往上100个版本写100个^比较容易数不过来，所以写成HEAD~100
    • git reset –hard HEAD^ 回退到上一个版本。。以此类推
    • 也可以用版本ID来回退git reset –hard 123。。
    • 电脑关机后，第二天后悔，需要回退，则需要命令git reflog查看命令历史。。。找到相应版本号的id之后，利用git reset –hard id 也可以回退
