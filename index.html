Git学习笔记

ubuntu下git的安装

  1. 旧版本的ubuntu linux的安装命令：sudo apt-get install git-core
    （我的linux是ubuntu 10.04，需要此条命令才能成功安装）

  2. 新版本的ubuntu只需要：sudo apt-get install git

  3. 以前老版本的Linux上有一个软件交git unit tools（安装命令是sudo apt-get install git并不是安装git代码仓库）。。所以只能使用git-core。之后才改为git。

  4. 基本命令：
    git init
    git add
    git status
    git commit -m “….”

  1. Git是分布式代码管理
  2. Git多平台
  3. Git是Linus（Linux之父的作品，大牛最开始是用C写的）
  4. Git的命令

    Git最好在Linux上使用

  • 建立Git工作目录:mkdir GitPrj
    目录下有文件_git(非常重要，不能轻易删除)
    • Git初始化： git init
    • 在目录下建立一个readme.txt
    • 添加需要提交的文本文件（Git只能管理文本文件，微软的wrod doc格式是二进制文件，不能管理）
      添加命令：git add reame.txt。。git可以添加多个文件，然后一次提交：比如：git add readme_1.txt readme_2.txt…readme_N.txt
    • 提交：git commit -m “提交说明”
    • git status查看git状态
    • git 查看文件先后的区别：git diff readme.txt
    • git 版本回退，首先查看提交日志：git log（完美显示加 –pretty=oneline）
    • HEAD表示当前版本
    • HEAD^表示前一个版本
    • HEAD^^表示前前一个版本
    • 当然往上100个版本写100个^比较容易数不过来，所以写成HEAD~100
    • git reset –hard HEAD^ 回退到上一个版本。。以此类推
    • 也可以用版本ID来回退git reset –hard 123。。
    • 电脑关机后，第二天后悔，需要回退，则需要命令git reflog查看命令历史。。。找到相应版本号的id之后，利用git reset –hard id 也可以回退

###设置黏滞位

Set uid, gid,sticky bit的三个权限的详细说明

一个文件都有一个所有者, 表示该文件是谁创建的. 同时, 该文件还有一个组编号, 表示该文件所属的组, 一般为文件所有者所属的组.

如果是一个可执行文件, 那么在执行时, 一般该文件只拥有调用该文件的用户具有的权限. 而setuid, setgid 可以来改变这种设置.

setuid: 设置使文件在执行阶段具有文件所有者的权限. 典型的文件是 /usr/bin/passwd. 如果一般用户执行该文件, 则在执行过程中, 该文件可以获得root权限, 从而可以更改用户的密码.

setgid: 该权限只对目录有效. 目录被设置该位后, 任何用户在此目录下创建的文件都具有和该目录所属的组相同的组.

sticky bit: 该位可以理解为防删除位. 一个文件是否可以被某用户删除, 主要取决于

该文件所属的组是否对该用户具有写权限. 如果没有写权限, 则这个目录下的所有文件都不能被删除, 同时也不能添加新的文件. 如果希望用户能够添加文件

但同时不能删除文件, 则可以对文件使用sticky bit位. 设置该位后, 就算用户对目录

具设置有写权限, 也不能删除该文件.

下面说一下如何操作这些标志:

操作这些标志与操作文件权限的命令是一样的, 都是 chmod. 有两种方法来操作,

1) chmod u+s temp — 为temp文件加上setuid标志. (setuid 只对文件有效)

chmod g+s tempdir — 为tempdir目录加上setgid标志 (setgid 只对目录有效)

chmod o+t temp — 为temp文件加上sticky标志 (sticky只对文件有效)

2) 采用八进制方式. 对一般文件通过三组八进制数字来置标志, 如 666, 777, 644等. 如果设置这些特殊标志, 则在这组数字之外外加一组八进制数字. 如

4666, 2777等. 这一组八进制数字三位的意义如下,

abc

a - setuid位, 如果该位为1, 则表示设置setuid

b - setgid位, 如果该位为1, 则表示设置setgid

c - sticky位, 如果该位为1, 则表示设置sticky

设置完这些标志后, 可以用 ls -l 来查看. 如果有这些标志, 则会在原来的执行标志位置上显示. 如

rwsrw-r– 表示有setuid标志

rwxrwsrw- 表示有setgid标志

rwxrw-rwt 表示有sticky标志

那么原来的执行标志x到哪里去了呢? 系统是这样规定的, 如果本来在该位上有x, 则这些特殊标志显示为小写字母 (s, s, t). 否则, 显示为大写字母 (S, S, T)

这三个权限的数字位可以这么理解

[root@server3 test]# 1 1 1

[root@server3 test]# rw s rws rwt

[root@server3 test]#

[root@server3 test]# SUID SGID Sticky

所以，可以得出

chmod 4777是设sid

chmod 2777是设置gid

chmod 1777是设sticky

test语句可测试三种对象：字符串、数值、文件属性，每种测试对象都有若干测试操作符
例如:
      test  "$answer"  =  "yes"
      变量answer的值是否为字符串yes
      test  $num -eq  18
      变量num的值是否为整数18
      test  -d  tmp    
      测试tmp是否为一个目录名
-------------------------------
test命令测试的条件成立时, 命令返回值为真(0)，否则返回值为假(非0).
      用法一：
                            test   $name  -eq   $1   
                            echo  $?
      用法二：
                            if   test   -f   $filename
                            then 
                                     ……
                            fi
通常简写为:       if  [   -f  $filename  ]

吃饭、睡觉、修电脑

吃饭、睡觉、修电脑

标签（空格分隔）： 硬件 修电脑

1.两滴502胶水引发的惨案

故事是这样的：
那天，Casio的表带断了（卡西欧这个坑爹货），屌丝买不起表带啊！所以就去超市买了一瓶502胶水来修理表带，然后不知道是咋的，居然thinkpad R400的笔记本显示器最右侧沾上了两滴胶水，刚开始还以为是两滴水。后来发现，真实难受啊。强迫症爆发！于是，关机，用小刀去刮显示器上的胶水。刚开始，效果不错啊。基本上把胶水全都刮了。然而，强迫症真的是毁灭世界。因为还有一点点毛丝，所以强迫症患者接着继续刮。这一刮，结果悲剧了，把偏光膜给刮花了。当时就停下了。第二天，看着电脑边上两点刮花的痕迹，还是难受啊（强迫症真实无解），又开始刮，好了，悲剧，彻底把偏光膜给刮掉了。于是，显示器被刮的那一块，只能显示白色，不能显示其他颜色（至于液晶显示的原理，可以自己百度啊。拆开发现，基本上就是，一层封住液晶的玻璃，玻璃上面贴着一层偏光膜，偏光膜用于滤光，原理上和《大学物理.光学》里面的偏光片基本是一致的。）。屏幕终于花了。

2.液晶显示的原理与构造

液晶显示器是由封住了液晶的玻璃，（液晶是一种半液态的东东，在无电场作用下，液晶分子的排列是没有规则的，只有在电场的作用下，液晶分子的排列才是有规则的，这样才能达到漏光透视的效果。）玻璃上贴了一层偏光膜，偏光膜的作用就是光学中的滤光作用。用于显示各种颜色。如果没有偏光膜，那就只能显示白色。在液晶玻璃片的背后有一个银色的反光板，当然这个反光板是用于反光的，给液晶提供更多的光。这样我们才能看见很亮的显示器。这个光的来源是装载液晶屏底部的灯管（液晶屏分为LCD和LED背光，这二者的区别，在于液晶屏的背光，也就是二者的光源的不同。LCD是采用荧光管提供光源，应该叫CCFL，神马阴极冷荧光什么的。其实就跟生活中的日光灯是一样的，日光灯的原理，高中就学过啊，就是荧光灯管在瞬间高电压的激发下，将灯管中的汞蒸气击穿导电，分子再撞击荧光分子，才能原子的能级跃迁，放出光能。大概就是这样的。其中最关键的是那个瞬间高压，采用的是镇流器，也就是电感。o(︶︿︶)o 唉，好多姿势。你看，其实修电脑还可以复习和学习到很多姿势呢。所以LCD屏幕需要高压条来提供瞬间高压来使得荧光灯管发光，如果屏幕不亮，很有可能是灯管或者高压条的问题，那个高压基本是在12V之上，用普通的万用表是测不出来的，因为频率太高，万用表的采样频率太低，所以测不出来。关于采样频率，这里又回想到了《信号与系统》里面的采样定理了（要复原一个信号，采样频率至少是被采样频率的2倍）。而LED的屏幕采用led，也就是发光二极管灯条，那个就需要另外的电压转换条来提供电压，LED当今有两种控制发光方法：1、恒流控制。2、恒压控制。一般采用恒流，恒流的更稳定）。

以上大概就是液晶显示器的基本原理了

3、换屏幕续集

因为这款电脑的显示器是14寸的16：10的比例，这种显示器都是老电脑才有的，现在的显示器都是14寸的16：9的比例，所以一般电脑的显示器还配不上。想到我舅舅有一个华硕的老本子，那个本子应该是16：10的显示器，然后拿来换上了。刚开始，可能是我太粗暴了。直接用洗洁精去洗液晶屏，然后没吹干就装上了，一开始就可以点亮的（一般的显示器都是有30脚的屏线，所以一般都是没多大的区别的，都基本兼容）。后面正常开机后，居然闪了几下，于是我机智地关机了。这下又好了。高压条上的状态灯不亮。可以正常开机，打着手电筒是可以看清电脑内容的，以为是高压条坏了，于是买了高压条换上，不亮。
